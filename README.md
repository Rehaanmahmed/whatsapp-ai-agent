# WhatsApp AI Agent

A WhatsApp AI Agent built using Node.js, Baileys, and Google Gemini.

## Features

- WhatsApp Integration
- AI-powered Conversations
- Function Calling / Tool Use
- Weather Tool
- Calculator Tool
- Reminder Tool

## Tech Stack

- Node.js
- Baileys
- Google Gemini API

## Setup

1. Clone the repository

```bash
git clone https://github.com/Rehaanmahmed/whatsapp-ai-agent.git
```

2. Install dependencies

```bash
npm install
```

3. Create a `.env` file

```env
GEMINI_API_KEY=YOUR_API_KEY
```

4. Start the bot

```bash
node index.js
```
